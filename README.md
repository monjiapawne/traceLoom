# traceLoom

**traceLoom** is a Python-based traceroute tool that enriches each hop with additional network context.

![Preview](preview1.png)

## Enrichments
- Latency  
- MAC Address  
- TCP Ports  
- DNS Records  
- OS (via Nmap)

## CLI Usage
```bash
./traceLoom -t <target>
```

or via the fully featured GUI.
